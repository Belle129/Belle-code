처음 시작할 때 코드가 저장되어 있는 파일을 생성 후 시작. 그 후 작업판에서 새 파일 만들기 하고 제목 지정 후 시작함.
메모장으로도 작업이 가능하지만 Visual studio code가 작업하기 수월하게 되어 있음. 시간이 지나면 달라질 수 있기 떄문에 이런 것도 서치해보면 좋음.
작업 중인 HTML파일은 웹브라우저 주소 칸에 드롭다운 or ctrl+O 사용하여 파일 불러오기 하면 브라우저에서 작업 중인 화면 확인 할 수 있음.
Visual studio code에서 저장(ctrl+S)를 해야 브라우저에 변경된 사항이 보임. 변경 후 저장이 되어 있지 않으면 파일에 흰색 동그라미 표시가 뜸.

br은 줄바꿈. 새로운 줄. 여러 번 써서 빈칸 줄을 늘릴 수 있음. 닫는 문법 없이 사용
p는 paragraph로 문단 구분, 문단 바꿈.
p를 사용해서 여백을 표현하고 싶으면 CSS 문법을 사용해야 함.
태그와 태그사이의 여백 문법: margin, 위쪽 여백: top, 여백 크기: 40px
p style="margin-top:40px;" 이런 식으로 나타내면 40픽셀만큼 위쪽에 여백이 생김.

strong은 굵게 표시 u는 밑줄 표시입니다.

img는 이미지 삽입할 때 사용, 이미지는 작업 중인 파일에 저장하여 주소를 옮김.
src는 원천으로 ="이미지 주소" 입력하여 이미지 첨부 가능함.
width를 사용하여 이미지 크기 조정 가능, "100% 또는 450 이런 식으로"
예시: img scr="back.jpg" width="100%"
참고 사항: unplash 들어가면 저작권 없는 좋은 이미지 사용 가능함.
